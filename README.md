# Prime Vector Wiki Portal\n\nA searchable, interactive documentation portal for the Prime Vector relational topology framework.\n\n## Core Principle\n\n`@.kernel = Only non-violating interaction persists here.`\n\n## Features\n\n- 📚 Markdown-based documentation\n- 🔍 Full-text search functionality\n- 📱 Responsive design\n- ⚡ Built with Next.js 14 + TypeScript\n- 🎨 NEON color scheme\n- 🔗 Knowledge graph exploration\n\n## Getting Started\n\n### Prerequisites\n- Node.js 18+\n- npm or yarn\n\n### Installation\n\n```bash\nnpm install\nnpm run dev\n```\n\nOpen [http://localhost:3000](http://localhost:3000)\n\n### Build for Production\n\n```bash\nnpm run build\nnpm start\n```\n\n## Deployment\n\nSee `azure-pipelines.yml` for Azure DevOps deployment configuration.\n\n## Technologies\n\n- Next.js 14\n- TypeScript\n- Tailwind CSS\n- Remark (Markdown)\n- Gray Matter (Front matter)
